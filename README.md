## Pre Requisites
Add `rotorS` and `mav comm` repositories in the src.
```bash
git clone https://github.com/ethz-asl/rotors_simulator.git
git clone https://github.com/ethz-asl/mav_comm.git
```

Install zbar
```bash
sudo apt-get install libzbar0
```
